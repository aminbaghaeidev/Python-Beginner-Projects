### Overview
This project is a simple game named "Rock-Paper-Scissors" with python programming language. With this you have this chance to play with your computer, which the system can decide randomly. This game also have some features such as score counter and continues till the user input n.

## Directory Structure
```
Rock_paer_scissors/
| src/
| |-- check_answer.py
| |-- game.py
| |-- main.py
|-- README.md
|-- requirements.txt
```

## Requirements
`Python 3.6+`

## How to Run
1. Make sure that you already downloaded the `python` on your PC.
2. Download the `repo file`.
3. then try to cd or drag and drop the file in your terminal for the `path`, then run the command below on your terminal.

```bash
$ python src/main.py
```

## Modules
- `src/main.py`: The main module so by running this you can use the game.
- `src/game.py`: The module which includes all the game logics and rules.
- `src/check_answer.py`: Check the answer which user enters for continue the game or not.
